GitHub Contribution Analyzer

Web Tool · HTML / JavaScript

A single-page web tool that pulls live data from any public GitHub repository via the GitHub REST API and generates a ranked breakdown of top contributors, PR activity, and issue statistics. Built to be hosted as a standalone HTML file with no dependencies.

Space-Filling Curves

Master's Dissertation · University of St Andrews, 2024

A master's thesis titled "A Walk Along a Space-Filling Curve" investigating the analytic properties of space-filling curves, investigated as the accumulators of iterated function systems. This work involved a substantial amount of programming done in Python, gathering data for a full-length research paper written in LaTeX. Both the paper and the code can be found here.

Carbon Nanospheres

Exploravision · National 2nd Place

Won second place nationally in the Exploravision science research paper competition (Grades 10–12) for "Using Carbon Nanospheres to Reduce Decoherence in Quantum Computing Systems." The paper investigates the electromagnetic stability of carbon nanospheres — a feature essential to practical quantum computing.
